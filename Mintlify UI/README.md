# Mintlify Landing Page Clone (HTML & CSS)

This project is a static clone of the **Mintlify landing page**, built using **HTML5 and CSS3**.  
The goal of this project was to recreate the structure, layout, and dark-themed UI of the official Mintlify website using only HTML and CSS.
